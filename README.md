# ⚡ TeslaWaze

**The best Waze experience for Tesla vehicles.**

A lightweight, Tesla-optimized web app that brings real-time Waze traffic, hazards, police, and accidents directly into your Cybertruck, Model 3, Y, or S browser.

## ✨ Features

- **Real-time Waze Map** — Live traffic, police, hazards, crashes, and road closures
- **Follow Mode** — Automatically follows your vehicle (customizable distance: ¼, ½, 1, or 2 miles)
- **Dark Mode Overlay** — Adjustable darkness slider for better night driving
- **Persistent Zoom** — Remembers your preferred zoom level
- **One-tap "My Location"** — Quick GPS centering
- **Deep Link Search** — Send destinations directly to Waze navigation
- **Tesla Touchscreen Optimized** — Large buttons and clean interface
- **Works on all Tesla models** (Cybertruck, Model 3, Y, S)

## 🚀 How to Use

1. Go to: `https://teslawaze.github.io/`
2. Allow location access when prompted
3. Enable **Follow** mode and adjust distance as needed
4. Use the **Dark Mode** slider for night driving
5. Tap **📍 My Loc** anytime to recenter

**Recommended Workflow:**
- Use **Tesla Navigation** for official routing + voice guidance
- Keep **TeslaWaze** open for superior hazard/police detection

## 🛠️ Tech Stack

- Official Waze Live Map Embed
- Vanilla HTML + Tailwind CSS
- Browser Geolocation API
- Hosted on GitHub Pages

## 📱 Tesla Browser Tips

- Works best in fullscreen
- Hard refresh (hold refresh button) after updates
- Allow location permission for best experience
- Use alongside phone Waze for audio alerts
